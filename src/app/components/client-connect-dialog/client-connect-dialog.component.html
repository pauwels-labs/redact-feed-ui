+- Why am I seeing this?

This website is Redact-enabled, which means it uses the Redact Client on your device to decrypt and display data on your browser.

If you are seeing this error it means the browser cannot communicate with your client to access encrypted data, and as a result cannot function properly.

+- Still having trouble?

Install Redact Client
To use this website you need to have the Redact Client installed and configured on this device. The Redact Docs will get you started. The Redact Client is not yet available for mobile devices.

Configure Client Port
The Redact Client port, which is provided to the Redact Client as a configuration value, must match the port provided here.

Enable CORS
